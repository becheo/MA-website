Um einen Versuch an dem Prüfstand durchführen zu können, muss dem Prüfstand eine Spannung übermittelt werden. Der vorzugebende Spannungsverlauf muss dabei in einer csv-Datei gespeichert werden. Jede Zeile in der Datei entspricht einem Spannungswert zu einem bestimmten Zeitpunkt. Die Samplerate am Prüfstand beträgt 250 Hz. Somit entsprechen 250 Werte in der csv-Datei dem Spannungsverlauf für eine Sekunde. Zudem muss die erste Zeile der Datei wie folgt aussehen: "time,voltage". Diese Überschift ordnet die Werte in der csv-Datei der entsprechenden Spalte hinzu. Auf der rechten Seite ist ein Beispiel dargestellt, welches die Struktur der Datei verdeutlichen soll.

time,voltage
0.0,0.0
0.004,0.0012
0.008,0.0024
0.012,0.0036
0.016,0.0048
0.02,0.0059
0.024,0.0071
0.028,0.0084
0.032,0.0096

Für die Erstellung des Spannungsverlauf kann das hier verfügbare Python Skript verwendet werden. In der Datei 'create_voltage_curve.py' ist dazu ein Beispiel gegeben. Beachten Sie, dass für die Verwendung des Skripts die Pakete "numpy" sowie "pandas" installiert sein müssen.

Python Skript herunterladen


Nachdem Sie einen Spannungsverlauf erstellt haben, müssen Sie diesen noch an den Prüfstand übergeben. Dafür dient diese Webseite. Erstellen Sie einen Account und loggen Sie sich anschließend ein. Nach dem Login sehen Sie ihr persönliches Dashboard. Hier können Sie ihre Messungen verwalten. Fügen Sie den erstellten Spannungsverlauf über den Button 'Neue Messung hinzufügen' ihrem Dashboard hinzu.

Sobald Sie ihren Spannugsverlauf hinzugefügt haben, können Sie diesen über den Button 'Messung starten' am Prüfstand ausführen. Über den Reiter 'Webcam' können Sie Ihre Messung live am Prüfstand verfolgen.

Sobald die Messung durchgeführt wurde, werden die Messdaten an der entsprechenden Stelle auf dem Dashboard angezeigt. Sie können die Messdaten inspizieren und diese als csv-Datei herunterladen. Mit den erhaltenen Daten können Sie dann die Identifikation der Parameter Ihres Simulationsmodells vornehmen.

Viel Spaß bei der Verwendung des Prüfstands.
